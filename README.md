# BBHT

Bug Bounty Hunting Tools is a script to install the most popular tools used while looking for vulnerabilities for a bug bounty program.
 
# Tools

- dirsearch
- JSParser
- knockpy
- sqlmap-dev
- Sublist3r
- wpscan
- webscreenshot
- Massdns
- Asnlookup
- Unfurl
- Waybackurls
- Httprobe
- Seclists collection




# Installing
- git clone https://github.com/yuxiaokui/bbht.git
- cd bbht
- docker build -t bbht .
